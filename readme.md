## Pantera Mu Web

Pantera Mu Web (PMW) é um projeto Open-Source desenvolvido com o intuito de adquirir conhecimento e experiência em algumas funcionabilidades do Framework Laravel 5. 

## Funções

<b>Ranking</b>
<ul>
<li>Kills</li>
<li>Resetes</li>
</ul>

<b>Painel</b>
<ul>
<li>Logar</li>
<li>Alterar Senha</li>
<li>Deslogar</li>
</ul>
